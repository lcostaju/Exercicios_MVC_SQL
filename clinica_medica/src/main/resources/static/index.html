Instruçoes

Para acessar a lista de consultas, clique no botão "Lista de consultas".

Para acessar a lista de pagamentos, clique no botão "Lista de pagamentos".

Para acessar a lista de pedidos de exame, clique no botão "Lista de pedidos de exame".

Só é possivel criar um pagamento ou pedido de exame dentro da tela de consulta
